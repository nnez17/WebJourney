Entity, Symbol & Emoji

Penjelasan singkat HTML Entity

HTML Entity adalah karakter khusus yang diwakili oleh kode numerik atau karakter yang tidak dapat ditampilkan secara langsung di web browser. Contoh:   untuk spasi tidak terlihat, © untuk tanda hak cipta.

± plus-minus sign

Daftar simbol matematika beserta entity-nya

  • ± untuk tanda plus/minus
  • × untuk tanda kali
  • ÷ untuk tanda bagi
  • √ untuk akar
  • ∞ untuk tanda tak hingga
  • ≈ untuk tanda sebanding
  • ≡ untuk tanda sama dengan
  • ≠ untuk tanda tidak sama dengan
  • ≤ untuk tanda kurang dari atau sama dengan
  • ≥ untuk tanda lebih dari atau sama dengan
  • ∀ untuk tanda integral
  • ∑ untuk tanda sigma
